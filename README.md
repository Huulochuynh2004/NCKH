# Federated Learning-based Intrusion Detection System for IoT

This project develops an Intrusion Detection System (IDS) using Federated Learning to preserve data privacy in IoT environments.

## Features
- Privacy-preserving training using Federated Learning
- Detect attacks: DoH, DDoS, Data Exfiltration
- Feature extraction with AutoEncoder
- Classification using Transformer

## Technologies
- Python
- TensorFlow / PyTorch
- Scikit-learn

## Dataset
- CIC-IDS / IoT datasets

## Project Structure
- FL_IDS_NCKH_banchuan.ipynb: Model training
- README.md: Project description
